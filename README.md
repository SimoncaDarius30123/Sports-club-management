# ⚽ Sports Club Management

## Description
This repository contains the components for a **sports club management application**. The main project structure is divided into three distinct and modular directories:

* **SportsClubManagement** — The primary code, acting as the **backend** or **server logic**.
* **Web-client** — The **web client** (frontend) interface.
* **Mobile-client** — The **mobile client** (mobile application) interface.

This structure facilitates independent development and deployment of the core business logic, the web interface, and the mobile application.
